# 🌸 Mother's Day Surprise Website 💖

An interactive and colorful Mother's Day themed website created using HTML, CSS and JavaScript to celebrate love, memories and gratitude.

The website features animated gift boxes that reveal scrapbook-style memory cards with photos and heartfelt messages. It also includes smooth animations, glowing gradients, heart interactions and a final celebration page with flowers and love notes.

## ✨ Features

- 🎁 Interactive animated gift boxes
- 💌 Memory cards with photos and messages
- 🌈 Animated gradient background
- ❤️ Heart-shaped desktop cursor
- 💖 Floating heart effects on mobile taps
- ✨ Rotating card reveal animations
- 💐 Final bouquet celebration screen
- 📱 Responsive design for laptop and mobile devices

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 📸 Project Purpose

This project was built as a creative frontend mini-project and a digital Mother's Day surprise to practice:
- frontend development
- animations
- responsive web design
- interactive UI effects
